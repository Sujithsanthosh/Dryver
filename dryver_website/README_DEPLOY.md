# How to Deploy Your Website

Your website files are ready in this folder (`dryver_website`).

### Option 1: The Easiest Way (Netlify Drop)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop this entire `dryver_website` folder onto the page.
3. Your site will be live instantly!

### Option 2: Using Command Line (Surge.sh)
1. Open a terminal in this folder.
2. Run: `npx surge` (you might need `cmd /c npx surge` on Windows).
3. Follow the prompts to create an account and deploy.

### Option 3: GitHub Pages
1. Upload this folder to a new GitHub repository.
2. Go to Settings > Pages.
3. Select the `main` branch and save.
